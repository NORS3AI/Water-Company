# Starfrost — GitHub Pages source

This folder is the GitHub Pages source for **https://NORS3AI.github.io/Water-Company/**.

GitHub Pages is configured to serve from the `main` branch, `/docs` folder.

## Files

- `index.html` — public landing page
- `assets/landing.css` — landing page stylesheet (self-contained, mirrors the WordPress theme tokens)

## How to update

Edit `index.html` and/or `assets/landing.css` directly on `main`. GitHub Pages will rebuild automatically within a minute or two of the push.

## Notes

This landing page is intentionally **standalone** — it does not depend on the WordPress theme files. That way the public preview keeps working even if the theme is being refactored.
